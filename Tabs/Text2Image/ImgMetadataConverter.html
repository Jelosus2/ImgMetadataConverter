Cache (boolean):
If true, the hashes of the hashes will be stored in cache to make the process faster.
SkipDuplicates ?
SkipDuplicates (boolean):
If true, only the converted image will be saved.
Warning: When not saving duplicates the file name won't start with converted-...,
if an error occurs the image will still be saved but with SwarmUI's original metadata
and the image will be saved in SwarmUI's output path ignoring the extension's one.
OutputDirectory (text):
The directory where the images with the metadata converted will be stored
If you wish to use the same path as the OuputPath in Server -> Server Configuration use [SwarmUI.OutputPath] as the value.
If you wish to use the same path as the Format of the OutPathBuilder in User -> User Settings use [SwarmUI.OutPathBuilder] as the value.
